# Smart Donation System - Frontend (Demo)
## Quick start
1. Install Node.js (16+)
2. Open frontend folder and run:
   ```bash
   npm install
   cp .env.example .env
   npm run dev
   ```
3. Open browser at the dev URL shown by Vite (usually http://localhost:5173)
4. Use the demo accounts seeded in backend:
   - donor@example.com / password
   - ngo@example.com / password
